reprogistry - Check npm package reproducibility

Usage:
  reprogistry <spec>       Check reproducibility for an npm package specifier
  reprogistry -p <purl>    Check reproducibility for a PURL

Options:
  -h, --help      Show this help message
  -v, --version   Show version number
  -p, --purl      Treat input as a Package URL (PURL)
  -j, --json      Output as JSON

Examples:
  reprogistry lodash@4.17.21
  reprogistry lodash@latest
  reprogistry -p pkg:npm/lodash@4.17.21
  reprogistry -j express@4.18.2
